# Tic Tac Toe GAME 🎮

This is a simple **Tic Tac Toe** game built using **HTML, CSS, and JavaScript**.  
It allows two players to play on the same device with a clean and interactive UI.

## 🛠️ Features
- Two-player gameplay (X and O)
- Win/draw detection
- Reset button
- Responsive layout

## 🚀 How to Run
1. Clone the repository or download the ZIP.
2. Open `index.html` in your browser.
3. Start playing!
